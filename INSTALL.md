# 🚀 Guia de Instalação Rápida - Upper Tone

## Pré-requisitos

- Python 3.10 ou superior instalado
- pip (gerenciador de pacotes do Python)

## Instalação em 3 Passos

### 1️⃣ Clone ou Baixe o Projeto

```bash
# Via Git
git clone <url-do-repositorio>
cd Upper-Tone

# Ou simplesmente baixe o ZIP e extraia
```

### 2️⃣ Instale as Dependências

**Opção A - Via requirements.txt (Recomendado):**
```bash
pip install -r requirements.txt
```

**Opção B - Instalação Manual:**
```bash
pip install python-docx customtkinter pillow
```

### 3️⃣ Execute o Aplicativo

```bash
python upperTone.py
```

ou

```bash
python3 upperTone.py
```

## ✅ Verificação da Instalação

Se tudo estiver correto, você verá uma janela moderna com:
- Título "Upper Tone"
- 3 seções numeradas
- Tema dark profissional

## 🐛 Problemas Comuns

### "ModuleNotFoundError: No module named 'customtkinter'"

**Solução:**
```bash
pip install customtkinter
```

### "ModuleNotFoundError: No module named 'docx'"

**Solução:**
```bash
pip install python-docx
```

### Python não encontrado

**Solução:**
- Certifique-se de que o Python está instalado
- Baixe em: https://www.python.org/downloads/
- Durante a instalação, marque "Add Python to PATH"

## 🎯 Primeiro Uso

1. Execute o programa
2. Clique em "Escolher Arquivo Musical"
3. Selecione um arquivo .docx ou .txt
